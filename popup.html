MyMory v0.5 α

What is MyMory?
A lightweight tool that turns any LLM conversation into a context-extender file (.mmr). Keep, share, or resume chats on any other model without hitting token limits.

v0.6 Roadmap (coming soon)

  • 🔧 WASM micro-LLM for smarter compression
  • 🧩 VS Code / Obsidian plugins
  • 🪐 IPFS auto-pin & NFT minting
Min-sys: Chrome 109+, 50 MB RAM free, no GPU required.
